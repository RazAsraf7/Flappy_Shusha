# Flappy Shusha

## Overview

Flappy Shusha is a simple arcade-style game inspired by the classic Flappy Bird.

The project demonstrates game logic implementation using Python and basic graphics programming.

---

## Features

* Player-controlled character
* Obstacle generation
* Score tracking
* Game loop mechanics

---

## Gameplay

The player controls the character and must avoid obstacles while trying to achieve the highest score possible.

---

## Technologies

* Python
* Game development library (e.g., Pygame)

---

## Running the Game

Clone the repository:

```bash
git clone https://github.com/RazAsraf7/Flappy_Shusha
cd Flappy_Shusha
```

Run the game:

```bash
python main.py
```

---

## Purpose

This project was created as a fun way to practice programming concepts such as loops, event handling, and game mechanics.
