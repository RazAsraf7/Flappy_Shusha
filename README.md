# Flappy Shusha

## Overview

Flappy Shusha is a simple arcade-style game inspired by the classic Flappy Bird.

The project was developed using the **Unity game engine** and **C#**, focusing on implementing basic game mechanics and player interaction.

---

## Technologies

* Unity Engine
* C#
* Windows executable build

---

## Gameplay

The player controls a character that must avoid obstacles while progressing through the game environment.

The objective is to survive as long as possible and achieve the highest score.

---

## Running the Game

Download or clone the repository and run the provided executable file.

```
My project.exe
```

No installation or additional dependencies are required.

---

## Purpose

This project was created as an experiment in game development using Unity and C#, focusing on gameplay mechanics and basic game design.
